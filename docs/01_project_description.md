# Project Description: Auto Highlight Generator

## Overview
Auto Highlight Generator is a Business Intelligence tool designed to simplify YouTube content consumption. By leveraging the YouTube Data API and Natural Language Processing (NLP), the system analyzes user comments and video transcripts to provide real-time insights.

## Problem Statement
Users often struggle to find key moments in long videos or understand the general public sentiment without watching the entire content. This project provides an automated way to "highlight" the most important parts and sentiment trends.

## Main Objectives
1. Provide a dynamic dashboard using Streamlit.
2. Perform Sentiment Analysis on YouTube comments.
3. Extract top keywords and video highlights automatically.
