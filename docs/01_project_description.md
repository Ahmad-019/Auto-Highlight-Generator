# 2. Abstract
The "Auto Highlight Generator" is a Business Intelligence tool designed to optimize video content consumption. By leveraging the YouTube Data API and Natural Language Processing (NLP), the system extracts emotional peaks from audience feedback to identify "Golden Moments." This approach transforms raw social data into actionable insights for content creators and marketers.

# 4. Business Intelligence Project Description and Objectives
* **Domain:** Digital Content Analytics & Business Intelligence.
* **Goal:** To automate the process of identifying high-engagement segments in YouTube videos using sentiment analysis.
* **Objectives:**
    1. Develop a high-performance interactive dashboard using Streamlit.
    2. Implement a hybrid sentiment analysis engine capable of processing Arabic and English text.
    3. Create a proprietary scoring algorithm to rank "Golden Moments" based on audience emotion intensity.
    4. Provide content creators with data-driven insights to improve content strategy and engagement.
