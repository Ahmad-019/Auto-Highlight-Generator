# BI Dashboard Design

## UI/UX Framework
The dashboard is built using **Streamlit**, featuring a custom **"Cherry Red"** design system implemented via CSS injection.

## Visualizations
1. **Emotion Breakdown:** A Plotly bar chart displaying the frequency of 6 core emotions (Funny, Happy, Sad, etc.).
2. **Engagement Timeline:** A dynamic area chart showing emotional intensity over the video’s duration.
3. **Interactive Cards:** "Golden Moment" cards that provide direct links to the most engaging timestamps in the video.
